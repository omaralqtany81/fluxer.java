# Commander Bot 
A powerful administrative bot built using the [fluxer.java](https://github.com/omaralqtany81/fluxer.java) library.

## 🚀 Features
- **Advanced Management**: Room configuration, role assignment.
- **Auto-Mod**: Built-in anti-spam and restricted content filtering.
- **Rank Engine**: Dynamic generation of military-style rank cards.
- **Leveling**: Native support for user experience tracking.
- **Interactions**: Full support for Slash Commands (e.g. `/set`, `/status`).

## 🛠️ Usage
1. Set your `FLUXER_TOKEN` environment variable.
2. Build with `mvn clean package`.
3. Run with `java -jar target/CommanderBot.jar`.
